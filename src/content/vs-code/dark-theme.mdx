## Usage

You can use any theme you like I'm using the [Night Owl](https://marketplace.visualstudio.com/items?itemName=sdras.night-owl) theme as the base theme then overriding the colors below in the VS Code Settings

1. Press `Ctrl + ,` or `âŒ˜ + ,` to go to VS Code settings.
2. Click on the `Open Settings (JSON)` icon in the top right corner. It looks like a file with a arrow in the top right corner.
3. Paste the following code anywhere in the `settings.json` file.

```json
"workbench.colorCustomizations": {
    "editor.background": "#000000",
    "editorGroupHeader.tabsBackground": "#000000",
    "sideBar.background": "#000000",
    "sideBar.dropBackground": "#000000",
    "statusBar.background": "#000000",
    "titleBar.activeBackground": "#000000",
    "titleBar.inactiveBackground": "#000000",
    "panel.background": "#000000",
    "activityBar.background": "#000000",
    "terminal.background": "#000000",
    "editorGutter.background": "#000000",
    "tab.inactiveBackground": "#000000",
    "toolbar.activeBackground": "#000000"
  },
```
